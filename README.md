#  Soonify

> **Organize. Collaborate. Succeed.** > The ultimate academic planner designed exclusively for college students.

##  Overview
**Soonify** is a full-stack web application that helps college students manage their hectic academic lives. It goes beyond a simple to-do list by offering features tailored for campus life, such as tracking exam schedules, managing assignments, and coordinating with peers.

The goal of this project is to reduce academic stress by centralizing dates and deadlines in one intuitive platform with real-time collaboration capabilities.

##  Key Features
* ** Event & Exam Scheduling:** Visual calendar to track classes, exams, and submission deadlines.
* ** Shared Reminders:** Create reminders that can be shared with classmates so no one misses a deadline.
* ** Collaboration Mode:** A dedicated space for study groups to plan meetups and track project milestones together.
* ** Responsive Design:** Fully accessible on desktop and mobile browsers.
